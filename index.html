Список команд, которые рассмотрели на первом занятии по работе с Git

git --version
Проверка текущей версии програмы.
git config --global user.name "Your Name"
Если вы никогда ранее не использовали git, для начала вам необходимо осуществить установку. Выполните команду, чтобы git узнал ваше имя.
git config --global user.email "your_email@whatever.com"
Если вы никогда ранее не использовали git, для начала вам необходимо осуществить установку. Выполните команду, чтобы git узнал ваш email.
git init
Создание репозитория, команду выполняем один раз для всего проекта.
git add
Индексирование созданных файлов и изменений.
git commit -m "комментарий"
Фиксирование всех сохраненных изменений.
git status
Проверка текущего состояния репозитория.
git log
Получение списка произведенных изменений.
git checkout Адрес коммита
Переключение между коммитами.
git remote add origin https/....
Создание связи между GitHub репозиторием и репозиторием на компьютере.
git push origin master
Данные будут загружены на репозиторий в GitHub.
